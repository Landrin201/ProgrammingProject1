

PROJECT TITLE: Programming Project 1, Exercise P1.3

PURPOSE OF PROJECT: The purpose of this project is to demostrate basic knowledge
of using BlueJ to program in java by creating an image using basic functions. The 
image is created by using java to generate individual lines of text which come
together to form an image. The image generated is a 5 square by 5 square tic tac toe board.

VERSION or DATE: Version 1, 31 March 2015

HOW TO START THIS PROJECT: To run the program right click on the TicTacToeBoardPrinter
class and select "void main(String[] args)". A menu will pop up- click ok. No parameters 
are needed.

AUTHORS: Marc Kuniansky

USER INSTRUCTIONS: No parameters are needed for this program. Running it will generate the 
desired image.
